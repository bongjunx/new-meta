# NEW META — 웹 텍스트 기반 턴제 RPG

브라우저에서 바로 실행되는 순수 HTML/CSS/JS 턴제 RPG 샘플입니다.
빌드 도구·서버·DB·인증 없이 동작하며, 진행 상황은 브라우저 `localStorage`에 자동 저장됩니다.

## 실행 방법

```bash
# 아무 정적 서버로 열면 됩니다 (예시)
npx serve .
# 또는
python -m http.server 8000
```

브라우저에서 `index.html`을 직접 열어도 동작합니다.

## 게임 요소

### 기본
- **직업 5종**: 기사 🛡️ / 도적 🗡️ / 상인 💰 / 마법사 🔮 / 검투사 ⚔️
- **스탯 6종**: HP / MP / ATK / DEF / 치명타 확률 / 치명타 피해
- **스킬**: 직업별 액티브 3종(쿨타임 1~5턴) + 궁극기 1종(쿨타임 10턴), 스킬별 전용 이펙트
- **공통 패시브 10종**: 레벨업마다 얻는 스킬 포인트로 습득
- **전투**: 턴제 전투, 버프/디버프, 도트(독·화상), 기절, 회피, 보호막, 치명타, 도망
- **사냥터 5곳**: 초록 평원 → 어둠숲 → 수정 동굴 → 화산 지대 → 마왕성 (레벨 제한)
- **몬스터 7종**: 슬라임/박쥐/버섯/유령/골렘/임프/드래곤 × 색상 변형 (지역별 총 20종)
- **드랍**: 장비(등급 5단계: 일반~전설, 랜덤 부가 옵션), 물약, 강화석, 골드
- **장비 5부위**: 무기/갑옷/투구/장갑/장신구
- **강화**: +20까지, 단계별 성공 확률 감소, 골드+강화석 소모
- **사망 패널티**: 골드 10% 손실 후 마을 부활
- **상점 / 여관**: 물약·강화석·다이아 구매, 휴식으로 완전 회복
- **희귀 몬스터**: 황금 슬라임 (골드 8배 + 다이아)

### v6 — 종언 지역 (Lv.1000) & 편의 기능
- **사냥터 23곳 (Lv.1~1000)**: 무너진 차원로(500~580) → 영겁의 심연 → 신멸의 전장 → 무한 회랑 → **종언의 옥좌(880~1000, 최종 보스: 종언 그 자체)**
- **환생 포인트 게이트**: 종언 지역은 누적 환생 포인트를 요구 (2,000 → 60,000) + 환생 횟수(25~45회) + 각성 스킬 수 — 최상위 유저 기준 최종 지역까지 약 2개월 이상의 장기 성장 설계
- **Lv.500 초과 경험치 급증**: 종언 구간 레벨링은 느리게, 환생 반복이 핵심 성장 축
- **가방 장비 비교**: 부위별 필터 + 장착 장비와 전체 스탯 비교 테이블 + 목록에서 주스탯 증감(▲/▼) 표시
- **자동 전투 스킬 금지 옵션**: 기본 공격만 사용 (설정은 계정에 저장)

### v5 — 몬스터 개성화 & 스킬 각성
- **몬스터 스프라이트 10종 추가**: 늑대·미라·해파리·폭풍조·눈알·사신·살아있는 별·크라켄·천사상·창세 코어 — 던전 테마에 맞게 전면 재배치 (설원=늑대, 유적=미라, 심해=해파리+크라켄, 일식=사신, 만신전=천사상 등)
- **✦ 스킬 각성**: 스킬 Lv.50 + 환생 3회 달성 시, 각성석 🌠(Lv.250+ 보스·탑 50층+ 보스 전용 드랍) + 💠500 + 🪙100만으로 스킬이 **완전히 새로운 형태로 변신** (직업별 4종, 총 20종의 각성 스킬)

### v4 — 500레벨 초월 지역
- **사냥터 18곳 (Lv.1~500)**: 별무리 잔해(200~250) → 공허의 바다 → 일식의 땅 → 용들의 영역 → 만신전 → **창세의 문(450~500, 최종 보스: 창조주의 그림자)**
- **환생·장비 게이트**: Lv.300+ 지역은 환생 횟수와 장착 장비 강화 합계를 요구 — 일식의 땅(환생 3회·강화 +50) → 창세의 문(환생 10회·강화 +100 = 5부위 전원 +20)
- **초고레벨 몬스터 벽**: Lv.250부터 추가 스케일 곡선 — 환생 스탯 스택 없이는 수치적으로 진행 불가
- **장비 티어 15단계**, 레벨 업적 Lv.500까지, 신규 몬스터 24종 (도감 업적 자동 연동, 총 138종)

### v3 — 200레벨 확장
- **사냥터 12곳 (Lv.1~200)**: 설원 → 고대 유적 → 심해 신전 → 하늘 섬 → 차원의 균열 → 신들의 정원 → 태초의 대지 (최종 보스: 태초의 용신)
- **액티브 스킬 강화**: 각 스킬 최대 Lv.100, 레벨당 위력 +2% — 골드 + 스킬의 서 📖 + 고레벨은 다이아까지 (비용 급증)
- **패시브 20종 × Lv.100**: 관통/피해감소/물약강화/도트증폭/보호막증폭/처치회복/경험치/강화확률/상점할인/궁극기증폭 등 신규 10종

- **📮 건의사항**: 유저 → 운영자 단방향 전송 (하루 10건), 관리자 페이지에서 열람

## 캐릭터 이미지 교체

`assets/` 폴더에 정해진 파일명으로 PNG를 넣으면 자동으로 적용됩니다.
자세한 내용은 [assets/README.md](assets/README.md) 참고. 이미지가 없으면 내장 픽셀 아트로 표시됩니다.

## 구조

```
index.html      화면 골격 (직업 선택 / 마을 / 전투)
css/style.css   전체 스타일 + 전투 연출 애니메이션
js/data.js      직업·스킬·패시브·몬스터·사냥터·장비 데이터
js/sprites.js   픽셀 아트 렌더러 + assets 이미지 오버라이드
js/game.js      코어 로직 (스탯 계산, 드랍, 강화, 저장)
js/battle.js    턴제 전투 엔진 + 스킬 이펙트
js/ui.js        화면 렌더링
js/main.js      초기화 / 이벤트 배선
```
