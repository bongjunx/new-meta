# NEW META — 웹 텍스트 기반 턴제 RPG

브라우저에서 바로 실행되는 순수 HTML/CSS/JS 턴제 RPG 샘플입니다.
빌드 도구·서버·DB·인증 없이 동작하며, 진행 상황은 브라우저 `localStorage`에 자동 저장됩니다.

## 실행 방법

```bash
# 아무 정적 서버로 열면 됩니다 (예시)
npx serve .
# 또는
python -m http.server 8000
```

브라우저에서 `index.html`을 직접 열어도 동작합니다.

## 게임 요소

- **직업 5종**: 기사 🛡️ / 도적 🗡️ / 상인 💰 / 마법사 🔮 / 검투사 ⚔️
- **스탯 6종**: HP / MP / ATK / DEF / 치명타 확률 / 치명타 피해
- **스킬**: 직업별 액티브 3종(쿨타임 1~5턴) + 궁극기 1종(쿨타임 10턴), 스킬별 전용 이펙트
- **공통 패시브 10종**: 레벨업마다 얻는 스킬 포인트로 습득
- **전투**: 턴제 전투, 버프/디버프, 도트(독·화상), 기절, 회피, 보호막, 치명타, 도망
- **사냥터 5곳**: 초록 평원 → 어둠숲 → 수정 동굴 → 화산 지대 → 마왕성 (레벨 제한)
- **드랍**: 장비(등급 5단계: 일반~전설, 랜덤 부가 옵션), 물약, 강화석, 골드
- **장비**: 무기/갑옷/장신구 장착, 판매
- **강화**: +15까지, 단계별 성공 확률 감소, 골드+강화석 소모
- **사망 패널티**: 골드 10% 손실 후 마을 부활
- **상점 / 여관**: 물약·강화석 구매, 휴식으로 완전 회복
- **희귀 몬스터**: 황금 슬라임 (골드 8배)

## 캐릭터 이미지 교체

`assets/` 폴더에 정해진 파일명으로 PNG를 넣으면 자동으로 적용됩니다.
자세한 내용은 [assets/README.md](assets/README.md) 참고. 이미지가 없으면 내장 픽셀 아트로 표시됩니다.

## 구조

```
index.html      화면 골격 (직업 선택 / 마을 / 전투)
css/style.css   전체 스타일 + 전투 연출 애니메이션
js/data.js      직업·스킬·패시브·몬스터·사냥터·장비 데이터
js/sprites.js   픽셀 아트 렌더러 + assets 이미지 오버라이드
js/game.js      코어 로직 (스탯 계산, 드랍, 강화, 저장)
js/battle.js    턴제 전투 엔진 + 스킬 이펙트
js/ui.js        화면 렌더링
js/main.js      초기화 / 이벤트 배선
```
